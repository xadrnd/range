# python3-seco-range
### Requirements
- Python 3
- [pysvn](https://pysvn.sourceforge.io/downloads.html)

### Usage
    $ python3
    Python 3.6.8 (default, Aug  7 2019, 17:28:10)

    >>> 
    >>> import seco.range
    >>> 
    >>> range_session = seco.range.Range(host='localhost')
    >>> range_session.expand('%{mycluster}:STABLE')
    ['hostname1', 'hostname2']
